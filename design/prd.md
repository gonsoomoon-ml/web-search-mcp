## 요구사항
Claude Cowork 는 Web Search 기능을 제공을 하고 있어.
내가 사용하고자 하는 Claude Code on Bedrock 은  Web Search 기능을 제고하고 있지 않아.
MCP Server 를 자체 구축해서 Claude Code on Bedrock 의 등록해서 사용을 하고 싶어.

## 리서치 요구사항
Claude Cowork 는 Web Search 와 Claude Code on Bedrock with MCP 서버 의 장, 단점 비교하고 싶어.
Claude Code on Bedrock with MCP 이 단점이 많으면 어떻게 기술적인 개선을 할 수 있을까?
